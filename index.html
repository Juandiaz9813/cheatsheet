CSS

Selectors
Description
Type Selector A type selector is sometimes referred to as a tag name selector or element selector because it selects an HTML tag/element in your document. Type selectors are not case-sensitive. In the example below, we have used the span, em and strong selectors.
The Universal Selector The universal selector is indicated by an asterisk (*). It selects everything in the document (or inside the parent element if it is being chained together with another element and a descendant combinator).
Class Selectors The case-sensitive class selector starts with a dot (.) character. It will select everything in the document with that class applied to it.
ID Selector The case-sensitive ID selector begins with a # rather than a dot character, but is used in the same way as a class selector.
